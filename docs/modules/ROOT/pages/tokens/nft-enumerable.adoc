= Non-Fungible Enumerable

Enumerable extension for xref:tokens/non-fungible.adoc[Non-Fungible Token] allows for enumeration of all the token IDs in the contract as well as all the token IDs owned by each account. This is useful for applications that need to list or iterate over tokens, such as marketplaces or wallets.
